# password-generator
A script to generate passwords

# Usage
```
usage: pass_gen.py [-h] [-l password_length] [-c capitals_amount] [-d digits_amount] [-s symbols_amount]

Generate a random password

optional arguments:
  -h, --help          show this help message and exit
  -l password_length  password length (default 8)
  -c capitals_amount  capitals letters amount (default 1)
  -d digits_amount    digits amount (default 1)
  -s symbols_amount   symbols amount (default 1)
```
